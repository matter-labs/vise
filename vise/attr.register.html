#[register]
Expand description

Registers a Global metrics instance or Collector, so that it will be included into registries instantiated using MetricsCollection.

This macro must be placed on a static item of a type implementing CollectToRegistry.

use vise::{Gauge, Global, Metrics};

#[derive(Debug, Metrics)]
#[metrics(prefix = "test")]
pub(crate) struct TestMetrics {
    gauge: Gauge,
}

#[vise::register]
static TEST_METRICS: Global<TestMetrics> = Global::new();
